# Project 1: OpenVAS Local System Scan

## Objective

To perform a basic vulnerability scan on my own system using OpenVAS and understand how scanning works.

---

## Steps Performed

1. Installed and configured OpenVAS
2. Accessed the web interface
3. Created target (local system)
4. Created scan task
5. Ran the scan
6. Viewed and exported results

---

## Results

* Total findings: 45
* Severity: Informational (Log level)
* No high, medium, or low vulnerabilities found

---

## Key Observation

* Multiple services detected (web, DNS, TLS)
* No critical vulnerabilities found
* System exposure identified through open ports

---

## Learning

* Learned how to perform vulnerability scanning
* Understood scan workflow (target → task → scan → report)
* Learned that scanning also reveals system exposure, not just vulnerabilities

---

## Disclaimer

This scan was performed on my own system for educational purposes only.
