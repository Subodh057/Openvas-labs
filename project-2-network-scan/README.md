# Project 2: Network Device Scan using OpenVAS

## Objective

To perform a vulnerability scan on another device in the same network using OpenVAS.

---

## Target

* Device: Friend’s Mac (authorized)
* IP Address: 192.168.1.90

---

## Steps

1. Identified target IP address
2. Added target in OpenVAS
3. Created scan task
4. Ran scan
5. Reviewed results

---

## Results

* Total findings: 1
* Severity: Low

---

## Key Finding

* TCP Timestamp Information Disclosure

  * The system allows uptime information to be calculated
